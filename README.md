# wiki-search-engine

### Tech Stack
 - React for frontend
 - Node for backend
 - SQlite + FTS5 for fzf
 - better-sqlite3
 - Fuse.js (If a better search is required)

## Purpose

A search engine for fandom wiki data parsed with [this wiki table parser](https://github.com/RyanMeline/Wiki-Table-Parser)
